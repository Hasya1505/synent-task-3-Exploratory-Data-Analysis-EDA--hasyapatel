The dataset contains approximately 16,000 records of Netflix titles.

Key attributes used in analysis:

Engagement metric: popularity
Time variable: release_year
Category fields: genres, country
People data: director, cast

The duration column is completely missing (100% null), so it is excluded from analysis.

👉 Instead, analysis focuses on:

Popularity (how much content is watched)
Vote count & rating (quality perception)

✔ Handling Missing Data
The duration column is ignored due to complete absence
Other columns are cleaned based on importance
✔ Date Standardization
date_added is converted into proper date format
Extracted useful components like year
✔ Genre Processing
Multi-genre values (e.g., “Comedy, Drama”) are split
Each genre is treated independently for accurate counting
✔ Duplicate Handling
Duplicate titles or IDs are checked and removed
Ensures no inflation in counts or trends
